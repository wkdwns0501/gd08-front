input 태그의 type

password : 입력되는 내용이 기호 처리


number : 숫자 입력


email : 이메일 형식


date : 날짜 형식 입력


color : 색상표 선택


hidden : 값은 있지만 화면에 보이지 않음


file : 파일 선택, 업로드시 파일 선택


button : 이벤트 처리로 기능 설정


submit : form 태그에 설정된 입력값을 action 속성의 값으로 페이지 전송


reset : form 태그의 입력된 값을 초기화


radio : 단일 선택, name 속성의 값으로 단일 선택
