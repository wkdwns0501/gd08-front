자바 스크립트 개요

HTML 페이지의 동작을 사용자의 요구에 맞도록 수행 가능
브라우저의 언어
인터프리터 방식의 언어

자바스크립트의 특징

  • 접근제한자 개념X
  • 변수 선언이 필요X, 초기화시 자료형 결정
  • 웹 환경에 특화된 언어 -> 보안 등의 이유로 기능 제한
  • 보안에 취약함 : 소스 외부 공개

사용법

  • 인라인 (inline) : 태그내에 간단히 소스코드를 추가
  • 내부 (internal) : html 파일 내에 script 태그로 소스코드 추가
  • 외부 (external) : 별도의 .js 파일로 소스코드를 추가

인라인 방식 (inline)

< 태그 on 이벤트명="자바스크립트소스()">
<a href="javascript:자바스크립트소스()">실행</a>


링크 태그에서 알림창 출력 링크 태그에서 콘솔 출력

내부 방식 (internal)

html 페이지내에 script 태그 영역안에 자바스크립트 소스 작성
script 태그의 위치 head, body내에서 어디든지 가능
