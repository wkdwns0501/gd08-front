        font-size : 글꼴 크기 (px, em, %)
  • 일반글꼴
  • px 테스트
  • em 테스트
  • % 테스트

font-weight : 글꼴의 굵기를 표현

        선택자 {
            font-weight : normal, bold, border, lighter, 100, 400, 900
        }
  • 원래굵기
  • 굵은 글꼴
  • 원래 굵기보다 굵게 (border)
  • 원래 굵기보다 얇게 (lighter)

font-variant : 영문 텍스트 문구를 작은 대문자로 변경

        선택자 {
            font-variant:small-caps;
        }
