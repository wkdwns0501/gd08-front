1. 속성 선택자

선택하고자 하는 요소내에 이미 생성된 속성을 활용하여 선택 선택자 뒤에 []를 이용하여 사용 

        선택자[속성 = 속성값] {...}     : 속성=속성값인 경우
        선택자[속성 ~ = 속성값] {...}  : 속성에 속성값이 포함된 태그
        선택자[속성 | = 속성값] {...}
        선택자[속성 ^ = 속성값] {...}
        선택자[속성 $ = 속성값] {...}
        선택자[속성 * = 속성값] {...}

2. 자손 선택자와 후손 선택자

많은 요소들이 중첩되어 있는 경우
자손 : 바로 하위 요소들
후손 : 하위 요소들 전부 

        자손 선택자 : >
        a > b {
            스타일 속성 : 값;
        }
        후손선택자 : 공백
        a b {
            스타일 속성 ㅣ 값;
        }

div의 자손이면서 후손

3. 동위 선택자 같은 레벨의 요소

동위 관계에 있는 뒤에 위치한 특정 요소 선택 

        * a요소 바로 뒤에 있는 b 요소 한개
        a+b {스타일속성 : 값;}
        * a 요소 바로 뒤에 있는 모든 b 요소 전부
        a~b {스타일속성 : 값;}

4. 반응 선택자

사용자의 움직임에 따라 적용되는 선택자 

        *특정 요소에 클릭되는 순간 스타일 반영
        선택자 : active {스타일속성 : 값;}
        *특정 요소에 마우스가 올라온 순간 스타일 반영
        선택자 : hover {스타일속성 : 값;}

5. 상태 선택자

요소의 상태에 따라 선택되는 태그 

        1. 체크된(checked)상태의 요소에 스타일 설정 - 선택자 : checked
        2. 초점(focus)에 맞춰진 input 요소에 스타일 설정 - 선택자 : focus
        3. 활성화/비활성화 된 요소에 스타일 설정 - 선택자 : enabled/disabled
