데이터 입력 구문

  • 변수 = [window.]confirm("질문내용")
  • 변수 = [window.]prompt("질문내용")
  • 변수 = 특정요소.속성(id, className, innerHTML, innerText)

window.prompt("질문 내용")

prompt 호출시 " 질문내용과 입력할 수 있는 텍스트 상자", "확인/취소" 버튼이 보여지는 브라우저의 알림창(대화창) 실행
확인버튼 클릭시 텍스트 상자에 입력된 값을 문자열로 반환
취소버튼 클릭시 null값 반환

데이터를 출력하는 구문

  • [window.]alert("브라우저의 확인창에 출력시킬 메세지")
  • [window.]console.log("브라우저의 콘솔창에 출력시킬 메세지")
  • document.write("브라우저의 화면에 출력 시킬 메세지")
  • 태그.innerHTML = "태그 내부의 출력시킬 메세지
